Smart Campus API

A JAX-RS REST API for the Smart Campus coursework. It manages rooms, sensors, and sensor readings through a versioned API under /api/v1.

Coursework Endpoints

GET/api/v1 - Discovery endpoint

GET/api/v1/rooms - Get all rooms

POST/api/v1/rooms - Create a room

GET/api/v1/rooms/{id} - Get a room by id

PUT/api/v1/rooms/{id} - Update a room

DELETE/api/v1/rooms/{id} - Delete a room

GET/api/v1/sensors - Get all sensors

POST/api/v1/sensors - Create a sensor

GET/api/v1/sensors/{id} - Get a sensor by id

DELETE/api/v1/sensors/{id} - Delete a sensor

GET/api/v1/sensors/{sensorId}/readings - Get sensor reading history

POST/api/v1/sensors/{sensorId}/readings - Add a sensor reading
